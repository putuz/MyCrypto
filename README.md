🚀 MyCrypto

MyCrypto is a cryptocurrency market tracking application built with SwiftUI and Binance API.

#Features<br>
  -Live cryptocurrency prices<br>
  -Realtime WebSocket updates<br>
  -Candlestick charts<br>
  -Multiple timeframes (1m, 5m, 15m, 1h, 4h, 1d)<br>
  -Coin search<br>
  -Coin detail page<br>
  -Modern SwiftUI UI<br>

#Technologies<br>
  -SwiftUI<br>
  -MVVM<br>
  -Async/Await<br>
  -URLSession<br>
  -WebSocket<br>
  -Charts Framework<br>
  -Binance REST API<br>
  -Binance WebSocket API<br>
